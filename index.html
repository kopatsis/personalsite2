About Me:

I am an aspiring software engineer and in my final year Computer Science student at Oregon State University. I am in the post-baccalaureate program here at OSU and am on track to recieve my BS in Computer Science in 2024. My best classes so far include Software Engineering I & II, Database Administration, and Cloud Application Development. Cloud Application Development and Software Engineering I allowed me to create the EZPZ URL Shortener, which is a URL shortener/QR code generator that includes analytics and Auth0 login. The backend is written in Go with Gin and hosted on Google Cloud while the frontend is a SvelteKit project hosted on Cloudflare pages at the URL shqrl.co.

I currently work as the E-Commerce Product Data Coordinator at LA Balloons, which has very effectively bolstered both my programming abilities and my self-starting nature. Two of my most important projects include creating internal applications in Go that both utilize PostgreSQL databases for product information and sales data, joined with Microsoft Excel as an output for non-programming end users. Some of my smaller yet equally helpful projects include an API bridge between Shopify and Dropbox for media and an SEO & product filter tagging system, used on thousands of products. Beyond just programming projects, I also manage the data for multiple online sales channels.

I also enjoy creating side programming projects with whatever spare time I can find. Utilizing The Odin Project, I have made a number of front-end creations including a real-time weather app, “Where’s Waldo” JavaScript game, and Tic Tac Toe with an unbeatable minimax AI. i9 Fitness, my [soon to be complete] Go-based fitness program that generates personalized HIIT workouts with no equipment via a user leveling system and user feedback, is what I consider to be my best project to date.
